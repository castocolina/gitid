<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| wizard write → four config artifacts + key files | identity.Generate writes the key; PersistAll writes the four artifacts — must be gated on a PASS test or explicit skip-confirm |
| copy action → system clipboard | The clipboard receives a key string; it must be the PUBLIC key only |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-05.6-14 | Information Disclosure | copying the PRIVATE key to the clipboard | mitigate | copy.go feeds clipboard.Copy ONLY the .pub line; no code path reads the private key; private path shown as faint text (D-13 locked); TestCopyNeverTouchesPrivateKey asserts this |
| T-05.6-15 | Tampering | PersistAll for an un-authenticated key | mitigate | PersistAll fires only after both prove phases PASS + Enter confirm; skip-and-write requires an explicit second `y` and surfaces a [warning] (FIX-CREATE-01) |
| T-05.6-16 | Tampering | write without backup | mitigate | PersistAll routes through the identity write path (filewriter backup); confirm gate before write (SAFE-01/02/03) |
| T-05.6-17 | Denial of Service | keygen/test goroutine panic | mitigate | runGenerateCmd and the ported prove cmds wrap goroutines in recover() |
| T-05.6-18 | Tampering | clipboard injection via crafted hostname | accept | atotto/clipboard dispatches the raw pubkey string to pbcopy/wl-copy/xclip with no shell interpolation (RESEARCH Security Domain) |
| T-05.6-SC | Tampering | go installs | accept | No new dependencies (RESEARCH Package Legitimacy Audit: N/A) |
