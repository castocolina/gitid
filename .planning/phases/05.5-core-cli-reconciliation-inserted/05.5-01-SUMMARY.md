## What Was Built

### Task 1: E2E Harness

`e2e/harness_test.go` with `//go:build e2e` provides three exported test helpers:

- **`SandboxHome(t)`** — creates a temp dir, sets `HOME=<dir>` via `t.Setenv`, returns the path. All binary writes land in the sandbox.
- **`BuildBinary(t)`** — runs `go build -o <tmpdir>/gitid ./cmd/gitid` with `cmd.Dir = repoRoot(t)`, returns the binary path.
- **`FakeSSHDir(t, mode)`** — writes a static fake `ssh` script via `os.WriteFile(..., 0o700)` (never `exec.Command("sh","-c",...)` per D-20), sets `GITID_FAKE_SSH_MODE=<mode>`, returns the dir for PATH prepend. Script handles `-G` branch first (ssh -G fixture), then switches on `GITID_FAKE_SSH_MODE` (pass/denied/timeout/default).
- **`repoRoot(t)`** — walks up from cwd until `go.mod` is found.

`Makefile` gained a `test-e2e: build` target running `go test -tags e2e -race -timeout 60s ./e2e/...`, appended to the `.PHONY` line.

### Task 2: A4 Stability Test (PASSES)

`internal/sshconfig/marker_roundtrip_test.go` contains `TestProviderMarker_RoundTripStable` which:
1. Builds a Host block body with `# gitid: provider=github` as the **last line** (after `IdentitiesOnly yes`).
2. Decodes via `ssh_config.Decode`.
3. Re-serializes with `cfg.String()`.
4. Decodes again.
5. Asserts: Host alias, Hostname (ssh.github.com), Port (443), and IdentityFile are byte-identical across both decodes; the provider comment substring survives in the serialized form.

**Result: PASSES** — Assumption A4 is proved. The marker design in Plan 02 is safe.
