# Phase 03.1 Plan 02: Baseline Render + Write Layer Summary

**One-liner:** Byte-stable baseline/url-rewrites/gitignore renderers with fixed section order + three filewriter-chokepoint writers (idempotent, foreign-preserving, floor-model include).

## What Was Built

Six new exported symbols and two supporting types added to `internal/gitconfig/baseline.go`:

### Types
- `BaselineConfig` — Tier-2 optional fields (AutoCRLF, Pager, ExtraColors, DiffColorMoved, MergeConflictStyle, InitDefaultBranch, IncludeAliases); Tier-1 keys are unconditional
- `URLRewrite` — one HTTPS→SSH insteadOf mapping (HTTPSPrefix, SSHPrefix)
- `DefaultBaselineConfig()` — all Tier-2 enabled with D-04 gist values
- `DefaultURLRewrites()` — big-three in fixed github/gitlab/bitbucket order
- `DefaultGitignorePatterns()` — 13 patterns; SC-2-locked first six in fixed order

### Renderers (Task 1)
- `RenderBaselineBlock(cfg BaselineConfig) string` — fixed `[core][push][pull][fetch][color][diff][merge][init][alias]` order; tab-prefixed keys; `TrimRight` no trailing newline; no `[user]` section; no map iteration; user-supplied Tier-2 strings validated via `validateValue` before emit
- `RenderURLRewritesBlock(rewrites []URLRewrite) string` — fixed github/gitlab/bitbucket order; `validateValue` panics on injection attempt; empty slice → empty string
- `RenderGitignoreBlock(patterns []string) string` — one pattern per line in slice order; byte-stable

### Writers (Task 2)
- `WriteBaselineFile(baselineFilePath string, cfg BaselineConfig, rewrites []URLRewrite) (string, error)` — `EnsureDir` + `ReplaceBlock(baseline)` + `ReplaceBlock/RemoveBlock(url-rewrites)` + `filewriter.Write`; zero `os.WriteFile`
- `WriteGlobalGitignore(gitignorePath string, patterns []string) (string, error)` — `ReplaceBlock(gitignore)` + `filewriter.Write`; foreign content preserved
- `WriteBaselineInclude(gitconfigPath, baselineFilePath string) (string, error)` — `PrependBlockIfNotFound("baseline-include", ...)` + `filewriter.Write`; floor model (block at TOP of `~/.gitconfig` on first write; in-place on re-run)
