# Phase 05.7 Plan 02: Adopter Core Implementation Summary

**One-liner:** `adopter.Adopt` (migrate + reference), `ListCandidates` (filepath.Glob with managedNames filter), and `MatchIdentityName` (suffix → email → ErrAmbiguousIdentity) fully implemented and GREEN; `filewriter.CopyFile` added as safe-write chokepoint for fragment adoption.

## What Was Built

### Task 1: Adopt migrate + reference, never-delete invariant

Replaced the Plan 01 RED stub body of `Adopt` with the real orchestration:

- **AdoptMigrate path:** validates `identityName` via `identity.ValidateName`, builds `destPath = <gitconfigDir>/.gitconfig.d/<name>`, calls `deps.CopyFile(sourcePath, destPath)`, then `deps.WriteIncludeIf(name, destPath, matches)`. Aborts before `WriteIncludeIf` if `CopyFile` fails (no partial includeIf).
- **AdoptReferenceInPlace path:** skips the copy, calls `deps.WriteIncludeIf(name, sourcePath, matches)` pointing at the original.
- **Never-delete invariant (D-05):** `Adopt` never calls `BackupAndRemove`; the test suite verifies this with a counter-based fake.
- **Backup collection:** every non-empty `backupPath` returned by `WriteIncludeIf` is appended to `AdoptResult.BackupPaths`.

Added `filewriter.CopyFile(src, dst string) (backupPath string, err error)` — reads src via `os.ReadFile` and routes through `filewriter.Write(dst, content, 0o644)`, inheriting the full backup+atomic+chmod guarantees of the safe-write chokepoint.

### Task 2: ListCandidates + MatchIdentityName

Implemented `ListCandidates(homeDir string, managedNames []string) ([]string, error)` from scratch:

- `filepath.Glob(homeDir/.gitconfig_*)` for raw candidates.
- Filters out paths under `~/.gitconfig.d/` (already-managed fragments).
- Derives the filename suffix after `.gitconfig_` and skips names in the `managedNames` set.
- Returns `[]string{}` (not nil) when no candidates exist.
- No dependency on `doctor.CheckOrphans` (premise correction REVIEWS.md #1 confirmed: `CheckOrphans` detects only orphaned SSH/gitconfig managed blocks and unreferenced keys — it never globs `~/.gitconfig_*`).

Implemented `MatchIdentityName(sourcePath string, knownNames []string, accountEmails map[string]string, readFragment func(string) (gitconfig.FragmentInfo, error)) (string, error)`:

1. **os.Lstat symlink guard:** rejects symlink candidates with a distinct error (not `ErrAmbiguousIdentity`) — T-05.7-02-02.
2. **Filename suffix match:** if the base name suffix is in `knownNames`, return it.
3. **Email fallback:** `readFragment(sourcePath)` → look up `GitEmail` in `accountEmails` map → return the identity name.
4. **Ambiguous:** returns `("", ErrAmbiguousIdentity)` when neither path resolves.
