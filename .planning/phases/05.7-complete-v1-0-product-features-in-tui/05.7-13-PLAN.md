| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-05.7-13-02 | Tampering | preview/write drift on the includeIf | mitigate | The live includeIf preview and the write both go through gitconfig.RenderIncludeIf / the gitconfig writers from the same buildCreateInput match list; no hand-built includeIf. |

| T-05.7-13-04 | Information disclosure | review leaking private key | accept | Screen 4 shows only public-side artifacts (Host block / includeIf / fragment summary / allowed_signers public line / ssh -G); the private key is never rendered. |
| T-05.7-13-05 | Repudiation | unclear what landed on disk | mitigate | Screen 4 is a read-only summary of the actual written artifacts + backup paths + ssh -G resolution, making the on-disk state explicit. |
| T-05.7-13-SC | Tampering | npm/pip/cargo installs | mitigate | No new third-party packages; stdlib + existing internal packages only. No install tasks. |
