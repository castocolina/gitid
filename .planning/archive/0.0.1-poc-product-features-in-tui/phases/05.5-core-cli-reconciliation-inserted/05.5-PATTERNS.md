```go
func extractProviderMarker(body string) string {
    const prefix = "# gitid: provider="
```

```go
    for {
        pre := deps.PreWrite(staged.FinalPrivatePath, in.Hostname, in.Port)
        printPreWrite(out, pre)
        if pre.Outcome == tester.PASS {
            // D-03: authenticated PASS → auto-persist, no extra prompt
            return true, false, nil
        }
        fp(out, "\n[r] retry  [s] skip-&-write (offline/upload-later)  [q] quit\n")
        choice := strings.TrimSpace(strings.ToLower(prompt(r, out, "Choice", "r")))
        switch choice {
        case "s", "skip":
            // D-05: require explicit typed confirm before persisting without PASS
            fp(out, "Warning: key has not authenticated. You must upload the key before using this identity.\n")
            if confirm(r, out, "Write config artifacts without authenticated PASS?") {

        default: // "r" or empty → retry

        }
    }
```

## Shared Patterns

### Deps seam (injectable functions)

**Source:** `internal/identity/identity.go` lines 119-148 (`Deps` struct), `cmd/gitid/add_test.go` lines 17-51 (`fakeDeps`)

**Apply to:** All new/modified orchestration code. Every external effect (write, test, generate) is a function field on a Deps struct. The cmd layer fills it from real packages; tests inject fakes. Never call `os.WriteFile` or `exec.Command` directly in a domain function.

```go
// Pattern: function-field deps struct
type Deps struct {
    Generate func(in CreateInput) (StagedKey, error)
    // ... other function fields
}
```

### `fp(out, ...)` for output

**Source:** `cmd/gitid/add.go` line 261

**Apply to:** All cmd-layer output. Never use `fmt.Fprintf(os.Stdout, ...)`. Always use `fp(out, ...)` or `fmt.Fprintf(out, ...)` so output is capturable in tests.

```go
func fp(out io.Writer, s string) {
    _, _ = io.WriteString(out, s)
}
```

### `prompt(r, out, label, defaultVal)` pattern

**Source:** `cmd/gitid/add.go` (used throughout `gatherCreateInput`)

**Apply to:** All new interactive prompts in `add.go` and `update.go`. When a non-interactive flag is present, skip the prompt and use the flag value:
```go
// Flag-or-prompt pattern (D-09):
nameVal := flagName // from Cobra flag
if nameVal == "" {
    nameVal = prompt(r, out, "Identity name", "")
}
```
