# Phase 05.5 Plan 03: Auth-gated Create-Flow (FIX-CREATE-01) Summary

Auth-gated create-new flow rewired: key is generated to ~/.ssh immediately, the
loop tests auth (PASS = auto-persist; skip+confirm = warned persist; quit = key
stays, no config). The old pre-test "Write all four artifacts now?" prompt is removed.

## Final Create-Flow Prompt Order

The create-new prompt sequence after this plan (important for plan 07 E2E harness):

```
Create mode: [1/2/3]        → selectMode
Identity name:              → gatherCreateInput
Git user.name:
Git user.email:
Provider (github/gitlab):
Host alias [name.provider]:
Hostname [ssh.github.com]:
Port [443]:
Match gitdir [~/git/name/]:
Passphrase (empty for none):
--- key pair written to ~/.ssh here (D-01) ---
--- .pub copied to clipboard + upload instructions shown ---
Pre-write connectivity test: $ ssh ...
  (if not PASS):
[r] retry  [s] skip-&-write  [q] quit
Choice [r]:
  (on [s]): Warning: key is not yet authenticated...
            Write config artifacts without authenticated PASS? [y/N]:
  (on [q]): Quit. Key kept at ~/.ssh/... Run 'gitid doctor' when ready.
--- (on persist): four artifact preview + backup paths + resolved test ---
```

## What Was Built

### Task 1: Domain layer split — RenderPreviews + PersistAll, Confirmed removed

**internal/identity/identity.go:**
- `Confirmed bool` removed from `CreateInput` (D-02/D-05: persist gate is dynamic)
- `RenderPreviews(in CreateInput, staged StagedKey) CreateResult` — pure preview
  builder using FINAL paths; calls `RenderHostBlock(…, in.Provider)` (Plan 02 sig);
  no writes, no clipboard, no network
- `PersistAll(in CreateInput, staged StagedKey, deps Deps) (CreateResult, error)` —
  exported four-writer sequence: PersistKey (when PrivPEM!=nil) → WriteSSH →
  WriteGitconfig → WriteFragment → WriteAllowedSigners → Resolved; returns backup paths
- `runPipeline` simplified: no `Confirmed` gate; always writes (for Reuse/Rotate
  which are confirmed-write paths); PreWrite Failure gate retained

**internal/identity/modes.go:**
- `AddAccount`: `Confirmed: true` removed from CreateInput literal
- `rotateInput`: `Confirmed: true` removed from CreateInput literal

**TUI callers fixed to compile:**
- `tui/addaccountform.go`, `tui/createform.go`, `tui/updateform.go`: `Confirmed: false` removed
- `tui/prove.go`: `in.Confirmed = true` removed; calls `identity.Create(input, deps.identity)` directly

### Task 2: CLI loop — runCreateLoop, auth-gated runCreateNew, buildDeps rewrite

**cmd/gitid/add.go:**
- `runCreateLoop(r, out, in, staged, deps) (persist, skipConfirmed, err)`:
  - Calls `deps.PreWrite(staged.FinalPrivatePath, …)` on each iteration
  - On `tester.PASS` → returns `(true, false, nil)` immediately (D-03, no prompt)
  - On `[s]` + typed `y` → prints "not yet authenticated" warning, returns `(true, true, nil)` (D-05)
  - On `[s]` + `n` → loop continues (skip declined)
  - On `[q]` → prints key path + doctor hint, returns `(false, false, nil)` (D-04)
  - On `[r]` / empty → loops (default retry)
- `runCreateNew` rewritten to new shape (D-01..D-06):
  - `gatherCreateInput` (no Confirmed prompt — D-02 removed)
  - `--dry-run`: synthetic staged key + `RenderPreviews` + return (no keygen — Q1)
  - `deps.Generate(input)` → writes key to `~/.ssh` directly
  - Copy `.pub` + print upload instructions
  - `runCreateLoop` → on persist: `identity.PersistAll` → backup paths + resolved test
- `buildDeps` Generate: writes private key to final `~/.ssh` path at 0600 via
  filewriter chokepoint (T-05.5-08); writes public key at 0644; `TempPrivatePath ==
  FinalPrivatePath` (D-01, no temp staging)
- `buildDeps` Cleanup: unconditional no-op (no temp dir to clean up)
- `gatherCreateInput`: `dryRun` param unused (signature kept for callers); old
  `in.Confirmed = confirm(…)` lines removed
- `printBackupPaths` helper: prints SSHBackup/GitconfigBackup/AllowedSignersBackup
