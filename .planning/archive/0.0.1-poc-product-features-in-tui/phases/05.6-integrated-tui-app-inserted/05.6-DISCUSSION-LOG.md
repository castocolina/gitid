### Save + re-test semantics
| Option | Description | Selected |
|--------|-------------|----------|
| Structural edits trigger the prove loop | Structural change → modal prove-before-write (test→PASS→confirm→write); cosmetic edits write after confirm. | ✓ |

**Notes:** Keeps the 5.5 match semantics (gitdir/URL/both, OR-applied) and the SAFE-* backup+preview+confirm discipline on every write.

| Option | Description | Selected |
