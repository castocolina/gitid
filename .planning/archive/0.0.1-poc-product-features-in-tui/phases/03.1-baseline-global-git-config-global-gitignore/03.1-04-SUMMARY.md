## What Was Built

### `cmd/gitid/baseline.go`

Thin Cobra handler with two entry points:

- `newBaselineSetupCmd()` / `runBaselineSetup()` — 7-step orchestration per UI-SPEC:
  1. Resolve home, build absolute paths
  2. Build default selections (DefaultBaselineConfig, DefaultURLRewrites, DefaultGitignorePatterns); zdiff3 gate via deps.GitVersionAtLeast
  3. ScanConflicts on existing ~/.gitconfig
  4. Print unified preview (baseline block, url-rewrites + blast-radius warning, gitignore, include + floor note, conflicts)
  5. Interactive Tier-2 opt-out prompt [Y/n] + per-rewrite deselect prompts [Y/n]
  6. --dry-run short-circuit (prints preview, returns without writing)
  7. Single confirm [y/N], then WriteGlobalGitignore → WriteBaselineFile → WriteBaselineInclude, then write summary

- `newBaselineShowCmd()` / `runBaselineShow()` — calls ReadBaselineState, prints installed/not-installed/incomplete per UI-SPEC (no === wrapper)

### `cmd/gitid/main.go`

Added the `baseline` group (inline, per plan):
```go
baseline := &cobra.Command{Use: "baseline", ...}
baseline.AddCommand(newBaselineSetupCmd())
baseline.AddCommand(newBaselineShowCmd())
root.AddCommand(baseline)
```

### `internal/deps/deps.go` (additive)

Added `GitVersionAtLeast(major, minor int) bool` — the zdiff3 git-version gate seam. Moved from a local cmd function to the internal/deps package to satisfy the acceptance criteria that cmd/ contains no exec.Command calls.

### `internal/gitconfig/baseline.go` (idempotency fix)

Added `bytes` import and idempotency skip to all three writers:
- `WriteBaselineFile`: if `bytes.Equal(composed, existing)` → skip write, return `("", nil)`
- `WriteGlobalGitignore`: same pattern
- `WriteBaselineInclude`: same pattern

This is the SC-1/SC-2 correctness requirement: second run creates zero backups and produces byte-identical files.

### `cmd/gitid/baseline_test.go`

6 test functions:
- `TestBaselineSetup_DryRun` — asserts preview header, blast-radius warning, dry-run message, zero file creation
- `TestBaselineSetup_DryRun_NoConflictsNote` — asserts no-conflicts note on first run
- `TestBaselineShow_NotInstalled` — empty-state copy when no artifacts exist
- `TestBaselineShow_Installed` — installed read-back: baseline: installed, keys, 3 rewrites, 13 patterns
- `TestBaselineShow_Incomplete` — incomplete state when only include block exists
- `TestBaselineSetup_Idempotency` — SC-1/SC-2: writes twice, asserts byte-identical files + no new backup

All tests hermetic via `t.TempDir()` + `t.Setenv("HOME", tmpHome)`.

## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 2 - Missing Critical Functionality] Idempotency skip in baseline writers**
- **Found during:** Task 2 (idempotency test failed: second run created 3 new backups)
- **Issue:** `filewriter.Write` always creates a backup when file exists, even if content is identical. The UI-SPEC "unchanged (no write needed)" contract and SC-1/SC-2 require no backup on a zero-diff re-run.
- **Fix:** Added `bytes.Equal(composed, existing)` check before `filewriter.Write` in all three writers (WriteBaselineFile, WriteGlobalGitignore, WriteBaselineInclude). When content is identical, skip write and return empty backup path.
- **Files modified:** `internal/gitconfig/baseline.go`
- **Commit:** ce5d789

**2. [Rule 2 - Missing Critical Functionality] Move git version check to internal/deps**
- **Found during:** Task 1 acceptance criteria (`grep -Ec 'os.WriteFile|exec.Command' cmd/gitid/baseline.go` must return 0)
- **Issue:** Initial implementation had a local `gitVersionAtLeast` function with `exec.Command` in cmd/. Plan requires cmd/ to stay exec-free and use the `internal/deps` seam.
- **Fix:** Added `deps.GitVersionAtLeast(major, minor int) bool` to `internal/deps/deps.go`; updated baseline.go to call `deps.GitVersionAtLeast(2, 35)`.
- **Files modified:** `internal/deps/deps.go`, `cmd/gitid/baseline.go`
- **Commit:** ce5d789
