<objective>
Build `internal/platform` and `internal/deps`: the OS/toolchain seam. `platform` parses
`ssh -Q key` to decide the key algorithm (ed25519 default; single deliberate fallback
ed25519→rsa-4096→ecdsa; per-OS install hint when none available), exposes the macOS
`UseKeychain` guard predicate, and is the mini-DOC-01 seam the Phase-4 doctor will
generalize. `deps` reports tool availability. This is shared infrastructure for the
keygen + ssh-config + clipboard slices.

Purpose: D-09 (probe + fallback) and D-14 (no-algo install guidance) without a hard
opaque failure, plus the OS-detection seam SSH-03's `Host *` block depends on.
Output: `internal/platform/platform.go`, `internal/deps/deps.go`, and their test files.
</objective>
